# Repositório criado para centralizar e documentar minhas automações desenvolvidas em **n8n**, com foco em demonstrar minhas habilidades em:

- automação de processos
- integrações entre APIs
- orquestração de fluxos
- manipulação de dados
- otimização operacional
- construção de soluções low-code/no-code com visão de negócio

## Objetivo

Este repositório funciona como um **portfólio técnico de automações**, onde organizo fluxos em JSON que representam soluções reais para diferentes cenários de negócio.

## Sobre os projetos

Aqui você encontrará automações em formato `.json`, exportadas do **n8n**, podendo incluir casos como:

- captura e tratamento de leads
- automações de CRM
- integrações com WhatsApp, e-mail e APIs
- enriquecimento de dados
- notificações e alertas
- fluxos internos de operação
- automações com IA
